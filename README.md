# Real-Time Chat App

[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)

A **fullstack real-time chat application** built with **Node.js, Express, Socket.io**, and **React**.  
Supports real-time messaging, multiple users, and a responsive frontend. Perfect for learning and deploying chat applications.

## Features

- Real-time chat using **Socket.io**  
- Multi-user support  
- Responsive **React frontend**  
- Separate **Node.js/Express backend**  
- Persistent chat (can be extended with database)  
- Easy to extend with rooms or private chats  

---

## Tech Stack

- **Frontend:** React, Vite, CSS  
- **Backend:** Node.js, Express, Socket.io  
- **Version Control:** Git & GitHub
